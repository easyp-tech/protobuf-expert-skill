# Protobuf Expert — EasyP CLI Skill

Agent skill for Protocol Buffers design and the [EasyP CLI](https://github.com/easyp-tech/easyp) toolkit (drop-in buf.build replacement).

## What it does

Helps developers write idiomatic `.proto` files, configure EasyP correctly, and follow protobuf best practices:

- Writing, reviewing, and refactoring `.proto` files
- Setting up and modifying `easyp.yaml` configuration
- Choosing lint rules and configuring code generation
- Managing proto dependencies
- Detecting and resolving breaking API changes
- Debugging lint errors or generation failures

## Install

```bash
npx skills add easyp-tech/protobuf-expert-skill
```
